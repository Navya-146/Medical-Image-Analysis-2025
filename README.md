Submitted By - Sparsh Sharma (DA24C028), Sharvasti Sarkar (DA24C018), Navya Garg (DA24C011)


Problem Statement - This project focuses on building a 3D semantic model for automatic segmentation of glioma sub-regions (WT, TC, ET) from multi-parametric MRI (T1, T1ce, T2, FLAIR) modalities. 
Dataset - The BraTS dataset (jjwal Baid, Satyam Ghodasara, Suyash Mohan, Michel Bilello, Evan Calabrese, Errol Colak, Keyvan Farahani, Jayashree Kalpathy-Cramer, Felipe C Kitamura, Sarthak Pati, et al. The rsna-asnr-miccai brats 2021 benchmark on brain tumor segmentation and radiogenomic classification. arXiv preprint arXiv:2107.02314, 2021)
